# My Copilot Instructions for TRP1 Assessment

## Communication Style
- Use clear, step-by-step explanations
- Include code examples when explaining concepts
- Ask clarifying questions if my request is ambiguous
- Provide multiple approaches when appropriate

## Problem-Solving Approach
- Break complex problems into smaller, manageable steps
- Explain the "why" behind solutions, not just the "how"
- Suggest best practices and industry standards
- Highlight potential pitfalls and edge cases

## Context Management
- Maintain conversation history context
- Reference previous solutions when relevant
- Clarify assumptions before proceeding
- Summarize progress periodically

## Format Preferences
- Use code blocks with proper language tags
- Include comments in generated code
- Structure explanations with headings and lists
- Separate concerns (e.g., separate logic, UI, data layers)

## Error Handling
- Help debug errors with specific guidance
- Explain error messages in simple terms
- Suggest debugging strategies
- Provide fallback approaches
